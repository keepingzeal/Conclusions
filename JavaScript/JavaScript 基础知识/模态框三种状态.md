- #### alert

  - ```
    alert("Hello");
    ```

    

- #### pompt

  - ```
    let age = prompt('How old are you?', 100);
    
    alert(`You are ${age} years old!`); // You are 100 years old!
    ```

    

- #### confirm

  - ```
    let isBoss = confirm("Are you the boss?");
    
    alert( isBoss ); // 如果“确定”按钮被按下，则显示 true
    ```
